Verilog HDL Basics

Syntax

Verilog HDL is not a programming language

output somename , anothername;

reg somename;

Module Definition

module modulename(var1 , var2 , var3);

[type] var1;
[type] var2;
[type] var3;

endmodule;

or

module modulename(
    [type] var1 , 
    [type] var2 , 
    [type] var3);

endmodule;

Calling Module


let's say we have module in another file as

module testmodule(var1,var2,var3);
input var1 , var2;
output var3;
...
endmodule

or we could write 
module testmodule(
    input var1,
    input var2,
    output var3);
...
endmodule

use module name to call it 

testmodule(thisfilevar , anothervar , outputvar);//order important

or 
testmodule( //.dot represents target file var definition so order not important
    .var3(outputvar),
    .var1(thisfilevar),
    .var2(anothervar)
);
